# A/B-тест email-рассылки

## Условие

- Контрольная группа: 621 покупатель из 6076 получателей.
- Тестовая группа: 745 покупателей из 6076 получателей.

Нужно определить, является ли рост конверсии статистически значимым.

## Метод

Используется двусторонний z-тест для двух долей с объединённой оценкой вероятности под нулевой гипотезой. Для доверительного интервала разницы долей применяется необъединённая стандартная ошибка.

## Результат

- контрольная конверсия: около 10.22%;
- тестовая конверсия: около 12.26%;
- абсолютный эффект: около +2.04 п.п.;
- z-статистика: около 3.56;
- двустороннее p-value: около 0.00037;
- 95% доверительный интервал: примерно от +0.92 до +3.16 п.п.

На уровне значимости 5% нулевая гипотеза отклоняется. Положительный эффект статистически подтверждён, однако его устойчивость на будущих рассылках требует мониторинга.
