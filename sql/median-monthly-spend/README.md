# Медианные месячные траты постоянных клиентов

## Задача

Для каждого месяца найти медиану суммарных месячных трат клиентов, которые совершали покупки во всех месяцах, присутствующих в данных. Встроенная функция медианы не используется.

## Логика

- Определяется полный набор месяцев в данных.
- Выбираются клиенты, активные в каждом из этих месяцев.
- Для каждого клиента рассчитываются месячные траты с учётом скидки.
- Значения ранжируются внутри месяца.
- Для нечётного количества строк берётся центральное значение, для чётного — среднее двух центральных.

Синтаксис рассчитан на PostgreSQL.
