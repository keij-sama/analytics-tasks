# Семидневный Retention Rate

## Задача

Для пользователей, активных 1 марта 2019 года, рассчитать долю вернувшихся пользователей на каждый день от D0 до D6.

## Данные

Файл `data/retention_cohort.csv` содержит:

- `device_id` — идентификатор пользователя;
- `session_time` — время сессии в UTC.

## Решение

Запрос в `solution.sql`:

1. приводит время сессии к дате;
2. фиксирует когорту пользователей первого дня;
3. формирует диапазон D0–D6;
4. считает уникальных вернувшихся пользователей;
5. делит их число на исходный размер когорты.

Синтаксис рассчитан на PostgreSQL.
