# Восстановление выручки по истории цен

## Задача

В таблице `transactions` отсутствует сумма операции. Известны товар, тип операции и время события. Цена восстанавливается из таблицы `price_history`, содержащей интервалы действия цен.

## Предполагаемая схема

- `transactions(item_id, action, event_timestamp)`;
- `price_history(item_id, price, valid_from, valid_to)`.

Запрос использует полуоткрытый интервал `[valid_from, valid_to)`. Для текущей цены допускается `valid_to IS NULL`. Покупки увеличивают выручку, возвраты уменьшают её.
