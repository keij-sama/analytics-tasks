# Ревью запуска мобильной функции

## Контекст

После запуска функции «Покупка в 1 клик» общая конверсия снизилась, а число ошибок выросло. Требовалось проверить исходный аналитический слайд и сформулировать вывод для руководителя.

## Основной вывод

Проблема локализована на Mobile:

- Desktop сохраняет конверсию 15% во все три месяца;
- Mobile снижается с 10.5% в мае до 5.0% в июне;
- ошибки Mobile одновременно растут со 180 до 3500;
- связь между релизом, ошибками и падением конверсии существенно убедительнее общей фразы о пересечении графиков.

Рекомендация: не масштабировать текущую версию функции до устранения мобильных ошибок и повторной проверки конверсии.
