# Агрегация запусков приложения

## Задача

Сформировать два отчёта:

- уникальные пользователи и сессии по платформе и источнику;
- уникальные пользователи по городам.

## Запуск

```bash
python solution.py data/sample_launches.csv --output-dir outputs/generated
```

Исходный файл запусков из тестового задания отсутствовал в архиве, поэтому для проверки скрипта добавлен небольшой синтетический пример. Файлы `outputs/original_*.csv` содержат результаты, полученные в исходной работе на полном наборе данных.

## Автоматизация

Скрипт принимает входной файл и каталог результатов через параметры командной строки, поэтому его можно вызывать из Airflow, cron или CI-процесса без изменения кода.
