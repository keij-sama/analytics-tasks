# Очистка цен в нескольких валютах

## Задача

Очистить строковые цены в USD и EUR, обработать пробелы, отрицательные значения, американские и европейские разделители, пропуски и некорректные строки. После очистки требуется рассчитать выручку по валютам и общую сумму в долларах.

## Запуск

```bash
python solution.py data/transactions.csv --output-dir outputs/generated
```

По умолчанию используется курс `1 USD = 0.87 EUR`. Его можно изменить параметром:

```bash
python solution.py data/transactions.csv --usd-to-eur 0.90
```

## Результаты

Скрипт сохраняет:

- `cleaned_transactions.csv` — очищенные строки и цена в USD;
- `revenue_by_currency.csv` — сумма исходных цен по валютам;
- `summary.json` — общая выручка в USD и количество отклонённых строк.
